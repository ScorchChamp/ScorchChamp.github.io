💪 365 Project challenge

Hey! 👋

I am Scorch, Welcome to my website! 😀

I am currently working on a 365 project challenge. The goal is to produce 365 simple applications, tools or generators; 1 for every day in 2023.

📃 Rules

  • 365 projects
  • 1 project per day
  • Working forward is allowed
  • Projects should be unique
  • All projects are open-source

⚙️ Techstack

💻 I try to create applications that can be hosted via github pages. That way I can make sure I can use the github API to keep the page's menu up to date

🚧 I also do some projects that cannot be hosted via github's static gh-pages environment. I try to make a github page for those projects with the result too!
